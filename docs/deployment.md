# Развертывание

## GitHub Pages

Этот сайт автоматически развертывается на GitHub Pages.

### Настройка

1. В настройках репозитория GitHub перейдите в раздел "Pages"
2. Выберите источник: "Deploy from a branch"
3. Выберите ветку: `main`
4. Выберите папку: `/ (root)`
5. Сохраните настройки

### Процесс развертывания

1. При пуше в ветку `main` автоматически запускается процесс сборки
2. GitHub Pages компилирует статические файлы
3. Сайт становится доступен по адресу https://dmitrycema.github.io

### Кастомные домены

Для использования кастомного домена:

1. Создайте файл `CNAME` в корне репозитория
2. Добавьте в него ваш домен
3. Настройте DNS записи у вашего провайдера

### Мониторинг

- Статус развертывания можно отслеживать во вкладке "Actions"
- Логи сборки доступны для каждого деплоя
