# dmitrycema.github.io

Личный веб-сайт и проектный репозиторий.

## Описание

Этот репозиторий содержит исходный код для личного веб-сайта, размещенного на GitHub Pages.

## Структура проекта

```
dmitrycema.github.io/
├── README.md           # Документация проекта
├── index.html          # Главная страница сайта
├── docs/              # Документация
├── assets/            # Статические ресурсы (CSS, JS, изображения)
└── .gitignore         # Исключения для Git
```

## Технологии

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Развертывание

Сайт автоматически развертывается на GitHub Pages при пуше в ветку `main`.

URL сайта: https://dmitrycema.github.io

## Разработка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/dmitrycema/dmitrycema.github.io.git
   ```

2. Внесите изменения в файлы

3. Отправьте изменения:
   ```bash
   git add .
   git commit -m "Описание изменений"
   git push origin main
   ```
